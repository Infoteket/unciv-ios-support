## Welcome

This site provides support and documentation for the iOS version of the open-source game Unciv.

- Official Unciv documentation: [https://yairm210.github.io/Unciv/](https://yairm210.github.io/Unciv/)
- Report issues or ask questions about the iOS build on the Unciv iOS issue tracker: [https://github.com/Infoteket/unciv-support/issues](https://github.com/Infoteket/unciv-support/issues)

Use the sidebar to navigate to Support topics and the Privacy Policy.
