## Support & Troubleshooting

This page collects iOS-specific notes, known issues, and guidance.

### About This Port

**Unciv iOS is a port of the Unciv project**, an open-source, moddability-focused remake of Civilization V made with LibGDX. The iOS version brings the same gameplay and features to Apple devices while maintaining compatibility with the main project.

**Release Schedule:** This iOS port follows the Unciv project's release schedule. The main project releases frequently as features and fixes become readyâ€”there is no fixed release cadence. When new versions of Unciv are released on GitHub, corresponding iOS updates will be published to the App Store. For the latest information about upcoming features and fixes, check the [Unciv GitHub releases](https://github.com/yairm210/Unciv/releases) page and join the [Unciv Discord community](https://discord.gg/bjrB4Xw).

### Installation

- Install the app from the App Store.

### Common Issues

- If you experience crashes or problems specific to the iOS build, please open an issue on the tracker with steps to reproduce: [Unciv iOS issues](https://github.com/Infoteket/unciv-support/issues)
- For gameplay questions, balance, or rules, consult the main Unciv documentation: [Unciv docs](https://yairm210.github.io/Unciv/)
- For features and compatibility questions, see the [Unciv GitHub repository](https://github.com/yairm210/Unciv)

### Multiplayer

Multiplayer games created from the iOS app are, by default, created on the Unciv project's multiplayer servers. See the Privacy Policy for details about server-hosted multiplayer. Multiplayer is compatible with the desktop and Android versions of Unciv.

### Contributing

If you want to help improve the iOS port, open issues or pull requests in this repository and include details on device, iOS version, and steps to reproduce any problems.

You can also contribute to the main Unciv project directly:
- Report bugs or suggest features on [GitHub](https://github.com/yairm210/Unciv/issues)
- Help translate the game into your language
- Contribute code or modding improvements
- Join the community on [Discord](https://discord.gg/bjrB4Xw)
