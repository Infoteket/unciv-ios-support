## Support & Troubleshooting

This page collects iOS-specific notes, known issues and guidance.

### Installation
- Install the app from the App Store. 

### Common issues
- If you experience crashes or problems specific to the iOS build, please open an issue on the tracker with steps to reproduce: [Unciv iOS issues](https://github.com/Infoteket/unciv-support/issues)
- For gameplay questions, balance, or rules, consult the main Unciv documentation: [Unciv docs](https://yairm210.github.io/Unciv/)

### Multiplayer
Multiplayer games created from the iOS app are, by default, created on the Unciv project's multiplayer servers. See the Privacy Policy for details about server-hosted multiplayer.

### Contributing
If you want to help improve the iOS port, open issues or pull requests in this repository and include details on device, iOS version, and steps to reproduce any problems.
